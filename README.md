

Ce dépôt GitHub rassemble l’ensemble de mes devoirs, exercices et projets réalisés dans le cadre de ma formation. Il inclut également plusieurs projets de réseau conçus avec Cisco Packet Tracer, mettant en œuvre des configurations telles que l’adressage IP, le routage, le DHCP, les VLANs et l’interconnexion d’équipements Cisco.


Remarque: Pour que vous reussi de regarder mes fichier faut telercharge le verion 7.3.1.0362 du Cisco Packet Tracer sur votre ordinateur.


## Compétences acquises

- Configuration de base des équipements Cisco 🖧
- Adressage IP et subnetting 🌐
- Mise en place du routage statique et dynamique 🔄
- Configuration du DHCP 📡
- Création et gestion de VLANs 🧩
- Interconnexion de routeurs et de switches 🔌
- Utilisation de Cisco Packet Tracer pour la simulation réseau 💻
- Analyse, test et dépannage de réseaux simples 🛠️
- Organisation et documentation de projets techniques 📚
